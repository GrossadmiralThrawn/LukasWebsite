Front-end

Android Studio offers a drag and drop GUI that allows you to drag and drop various UI elements onto a page internally known as an activity, where they are then arranged accordingly and can be customised to suit the project. Possible adjustments mainly concern the size, colour and possible text content. However, the number of customisation options is so large that anyone can use it to create their own very individual UI.
